# i18n — 多言語対応

Sphinx は `gettext` ベースの国際化機能を持っています。日本語ソースから英語バージョンを作るなどに使えます。

## 基本の流れ

```{mermaid}
flowchart LR
    A[".md / .rst"] --> B{{sphinx-build gettext}}
    B --> C[".potファイル"]
    C --> D{{sphinx-intl update}}
    D --> E[".poファイル"]
    E --> F[[翻訳者が編集]]
    F --> G{{sphinx-build}}
    G --> H["翻訳済み HTML"]
```

## セットアップ

```bash
pip install sphinx-intl
```

```python
# conf.py
locale_dirs = ["locales/"]
gettext_compact = False
```

## 翻訳ファイルの生成

```bash
# 1. .pot ファイルを生成
sphinx-build -b gettext docs/source docs/build/gettext

# 2. .po ファイルを生成（英語版の例）
sphinx-intl update -p docs/build/gettext -l en

# 3. docs/source/locales/en/LC_MESSAGES/*.po を編集して翻訳する

# 4. 英語版をビルド
sphinx-build -b html -D language=en docs/source docs/build/html-en
```

## 演習

:::{exercise} 演習 8-2: 英語版を作成する

1. `sphinx-build -b gettext docs/source docs/build/gettext` で .pot ファイルを生成する
2. `sphinx-intl update -p docs/build/gettext -l en` で .po ファイルを生成する
3. `docs/source/locales/en/LC_MESSAGES/index.po` を開き、トップページのタイトルを英語に翻訳する
4. `sphinx-build -b html -D language=en docs/source docs/build/html-en` で英語版をビルドする
:::
